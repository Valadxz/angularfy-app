## :toolbox: Para empezar

### :running: Correr de forma local
Clonar el API del proyecto

```bash
https://github.com/Valadxz/api-angularfy.git
```
Instalar dependencias
```bash
npm install
```
Iniciar el servidor
```bash
npm start
```

Clonar el proyecto

```bash
https://github.com/Valadxz/angularfy-app.git
```
Instalar dependencias
```bash
npm install
```
Iniciar el servidor
```bash
ng serve -o
```
